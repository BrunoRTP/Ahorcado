Para iniciar el juego tienes que ir a la terminal e instalar Node.js (si no lo tienes) o verificar la versión utilizando el comando node -v. Después tienes que utilizar el comando
json-server --watch palabras.json. Una vez hecho eso, puedes ya con Live Server cargar la página. Una vez cargado, arriba a la derecha saldrán las categorías. Para empezar el juego
tienes que darle a una de las letras. Tienes 10 segundos para darle a la siguiente letra. Tienes 6 intentos para adivinar la palabra; si fallas, pierdes. Una vez adivinada o fallada
la palabra, tienes que darle a "Reiniciar el juego" para volver a jugar. Otra forma de jugar es entrando por el enlace de Pages, el cual te deja jugar directamente.
